- when `--ph-h: 56px` (default) → `calc(56px - 24px) = 32px` (== current `top-8`) ✓ no visual change on routes that haven't opted into the composition
- when `--ph-h: 110px` (Coverage's title+toolbar) → `calc(110px - 24px) = 86px` (family-header slides down 54px)

- when `--ph-h: 56px` → `calc(56px + 25px) = 81px` (== current `top-[5.0625rem]` which is `81px`) ✓
- when `--ph-h: 110px` → `calc(110px + 25px) = 135px`

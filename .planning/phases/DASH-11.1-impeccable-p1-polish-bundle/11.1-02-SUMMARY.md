## Decisions Made

- **gitNexus `w-32` → `w-36`** (PD-11.1-06 gate applied): Pre-colgroup measured max 131px > 124px threshold. See GitNexus measurement section above.
- **`Object.freeze()` in addition to `as const`** — `as const` is TypeScript-only (compile-time readonly). `Object.freeze()` makes `Object.isFrozen()` return `true` at runtime, allowing the test to assert actual immutability rather than just the TypeScript contract.
- **Consumer-grep via `readFileSync`** — avoids jsdom/React render for a structural assertion. The test asserts code shape (import present, colgroup present, table-fixed present), not behavior. Faster, no environment dependency.

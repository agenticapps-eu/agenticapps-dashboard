<objective>
Wire the env.json store into the user-facing CLI and the daemon boot, and fold read-only Infisical scope reflection into the existing integrations route. Three deliverables: (1) `agentic-dashboard env set/list/unset` (allow-listed, 0600, redacted output); (2) `loadEnvFile()` called at boot under a try/catch so a corrupt file never blocks start; (3) INFI-03 — extend the integrations response value with `infisicalWorkspaceId` + `infisicalEnvironment` when `.infisical.json` is present-valid.

Output: envCmd.ts + test, cli.ts command wiring, start.ts boot call, integrations.ts INFI-03 extension + test.
</objective>

## PanelContainer Signature Extension

New prop added to `PanelContainerProps`:

```typescript
/** D-6.1-02: when true, collapses body until header click. Defaults to false. */
defaultCollapsed?: boolean
```

When `defaultCollapsed` is `true`:
- Header becomes a `<button>` with `aria-expanded={!collapsed}` and `aria-controls={panelId}-body`
- Body wrapped in `<div id="{panelId}-body" role="region">` — absent from DOM when collapsed
- ChevronRight shown when collapsed, ChevronDown when expanded — **no animation, no transition**
- Stale pill and unreachable banner remain visible in both states
- State resets per page-load (no localStorage)

When `defaultCollapsed` is `false` or omitted — identical to pre-06.1 behavior (static header, body always visible, no button).
